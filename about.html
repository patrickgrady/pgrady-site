Bio

Patrick Grady loves Star Wars, drawing, and has been designing for print and screen since earning a BFA in Graphic Design from the University of Illinois at Urbana-Champaign in 2001. Most recently exploring his passion for design systems as they relate to enterprise-level software as UI Design Manager for Centro. Currently seeking new opportunities: LinkedIn.

Colophon

This site was typeset in (the excellent!) Tiempos Headline and Text by Klim Type Foundry. It was built with Jekyll, and hosted on Github Pages. Gulp, Node-Sass, and Browsersync were used to create a quick-compiling build script that includes live updates on multiple devices. See the Project Starter Kit below for a look under the hood, it installs Breakpoint and Modular Scale without the need for Compass. Check out the style guide.
